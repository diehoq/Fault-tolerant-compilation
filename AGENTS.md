# Repository Notes

This repository is being built as a pedagogy-first reference implementation for concatenated Steane-code memory simulations.

Working expectations:

- Prefer explicit code over compact abstractions.
- Document modeling assumptions near the code that uses them.
- Add or update tests whenever behavior changes.
- Keep explanations accessible to a reader who wants to learn the implementation line by line.
- Avoid claiming exact historical fidelity to a specific fault-tolerant protocol unless the code and docs justify that claim directly.
